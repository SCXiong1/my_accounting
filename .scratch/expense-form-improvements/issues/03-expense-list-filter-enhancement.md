## What to build

支出记录页添加时间范围筛选，并集成分类标签联动筛选。

**时间筛选**：
- 顶部添加时间选择器按钮组（复用 usePeriodFilter composable）
- 支持预设时间段：本月、近3月、近6月、今年
- 支持自定义时间段（选择开始和结束月份）
- 时间范围作为查询参数传给后端 API

**筛选联动**：
- 集成 #2 的分类标签联动逻辑到筛选器
- 时间、分类、标签三个条件 AND 组合
- 从统计页面点击分类/标签跳转时，自动带入筛选条件

**后端**：
- 支出列表 API 支持 start_time、end_time 查询参数
- 筛选条件组合：时间 AND 分类 AND 标签

## Acceptance criteria

- [ ] 支出记录页顶部显示时间选择器，支持预设和自定义时间段
- [ ] 选择时间范围后，列表只显示该时间段内的支出
- [ ] 选择分类后，标签筛选器联动显示该分类下的标签
- [ ] 选择标签后，分类筛选器联动显示该标签用过的分类
- [ ] 时间、分类、标签三个条件可以同时生效（AND 逻辑）
- [ ] 从统计页面点击分类/标签跳转时，自动带入筛选条件
- [ ] 清空某个筛选条件不影响其他条件
