## What to build

回收站页面添加批量选择删除功能。

**前端**：
- 页面添加"批量删除"按钮，进入/退出选择模式
- 选择模式下每条记录显示复选框
- 底部显示已选数量和"删除"按钮
- 点击删除弹出确认对话框，显示将删除的记录数量
- 确认后调用批量删除 API，成功后刷新列表

**后端**：
- 新增批量永久删除 API（POST /v1/expenses/batch-delete）
- 接收 expense ID 列表，校验归属权，执行永久删除
- 返回成功删除的数量

## Acceptance criteria

- [ ] 回收站页面有"批量删除"按钮，可进入选择模式
- [ ] 选择模式下每条记录显示复选框
- [ ] 底部显示已选数量和"删除"按钮
- [ ] 点击删除弹出确认对话框，显示"确定永久删除 N 条记录吗？"
- [ ] 确认后批量删除，列表自动刷新
- [ ] 删除成功后显示提示
- [ ] 未选择任何记录时，删除按钮禁用
